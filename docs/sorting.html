Sorting

从项目元素中收集数据并在布局中重新排列它们的顺序。Collect data from item element and rearrange their order in the layout with sorting.

See Demo: Sorting

Markup

任何类似的项目组都有自己的数据。它可能是一个文本值，像一个标题或标记，或一个数值，像一个测量或等级。举例来说，每一项元素都有几个数据点，可用于排序。如元素的符号、数字、元素名称、重量和类别。Any group of similar items have their own data. It could be a text value, like a title or tag, or a numerical value, like a measurement or grade. For our example, each item element has several data points that can be used for sorting. There’s the elemental symbol, number, name of the element, weight, and category.

<div id="container">
  <div class="element transition metal" data-category="transition"> 
    <p class="number">79</p> 
    <h3 class="symbol">Au</h3> 
    <h2 class="name">Gold</h2> 
    <p class="weight">196.966569</p> 
  </div> 
    
  <div class="element metalloid" data-category="metalloid"> 
    <p class="number">51</p> 
    <h3 class="symbol">Sb</h3> 
    <h2 class="name">Antimony</h2> 
    <p class="weight">121.76</p> 
  </div>
</div>

getSortData option

为了从元提取这些数据，我们需要传递一个函数可以通过getsortdata选项。此选项接受一个对象，其值是提取数据的函数。In order to extract this data from the element, we need to pass in a function to get it via the getSortData option. This option accepts an object, whose values are the functions to extract the data.

每个函数接受一个参数，它代表一个jQuery对象中每一项元素。有了这个参数，函数需要返回数据点。Each function receives one argument, which represents a jQuery object for each item element. With that argument, the function needs to return the data point.

在上面的例子中，要获得元素名称，我们需要从“名称元素”中获取文本。符号相同的作品。In the example above, to get element name, we would need to get the text from the .name element. The same works for symbol.

$('#container').isotope({
  getSortData : {
    name : function ( $elem ) {
      return $elem.find('.name').text();
    },
    symbol : function ( $elem ) {
      return $elem.find('.symbol').text();
    }
  }
});

对于数值数据，我们可以把一个文本值转换成数字，使用parseInt()或者parseFloat()方法。For numerical data, we can convert a text value into a number with parseInt() or parseFloat().

getSortData : {
  // ...
  number : function ( $elem ) {
    return parseInt( $elem.find('.name').text(), 10 );
  },
  weight : function ( $elem ) {
    return parseFloat( $elem.find('.weight').text() );
  }
}

提取的数据可以是任何可以通过jQuery的项目元素。提取数据的类别属性的范畴内举行的数据，我们可以使用attr()方法。The data extracted can be anything accessible in the item element via jQuery. To extract the category data held within the data-category attribute, we can use the .attr().

getSortData : {
  // ...
  category : function ( $elem ) {
    return $elem.attr('data-category');
  }
}

获得灵感！Get creative! 您可以按每个项目元素的宽度在索引页中排序列表。You can sort the list in the index page by the width of each item element.

getSortData : {
  // ...
  width : function( $elem ) {
    return $elem.width();
  }
}

sortBy option

对每一个设置了getsortdata的方法，Isotope使用getSortData方法来建立数据的排序。数据缓存是在初始化时建立，所以它可以在排序时被快速访问。在上面的方法，我们已经建立了一个包括项目元素名称、符号、数字、重量和类别的数据。For every method set in getSortData, Isotope uses that method to build the data for sorting. The data cache is built on initialization so it can be quickly accessed when sorting. With the methods above, we have built data for an item elements name, symbol, number, weight and category.

元素排序是被sortBy选项处理。值需要匹配getsortdata对象的属性的名称。Sorting elements is done with the sortBy option. The value needs to match the property name used in the getSortData object.

With our example, we can use 'name', 'symbol', 'number', 'weight' and 'category'.

$('#container').isotope({ sortBy : 'symbol' });

Isotope建立了一个额外的排序数据“原始顺序”。“原始顺序”的排序将使用项目元素的原始顺序排列它们的布局。There is an additional sorting data built in to Isotope 'original-order'. Sorting with 'original-order' will use the original order of the item elements to arrange them in the layout.

sortAscending option

默认情况下，Isotope使用升序进行数据排序。By default, Isotope sorts data in ascension. If our data for name is “Gold, Antimony, Lead, Iron, Silver”, when sorted by name, the elements will be ordered ABC.. : “Antimony, Gold, Iron, Lead, Silver.” 扭转顺序为倒序排列，需要设置sortAscending为false。To reverse the order and sort data in descension, set sortAscending to false.

$('#container').isotope({ 
  sortBy : 'name',
  sortAscending : false
});

创建互动按钮。 Creating interactive buttons

We can use a simple list for our buttons.

<ul id="sort">
  <li><a href="#name">name</a></li>
  <li><a href="#symbol">symbol</a></li>
  <li><a href="#number">number</a></li>
  <li><a href="#weight">weight</a></li>
  <li><a href="#category">category</a></li>
</ul>

When one of these links is clicked, we can use the href attribute as the value for sortBy in the Isotope script.

$('#sort a').click(function(){
  // get href attribute, minus the '#'
  var sortName = $(this).attr('href').slice(1);
  $('#container').isotope({ sortBy : sortName });
  return false;
});
